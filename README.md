# Fixd Messaging

[![Android CI](https://github.com/fixdfun/fixd-messaging/actions/workflows/android.yml/badge.svg)](https://github.com/fixdfun/fixd-messaging/actions/workflows/android.yml)

**Fixd Messaging** is an open-source Android SMS/MMS client built by [Fixd.fun](https://fixd.fun)
for people who want a clean, privacy-respecting alternative to Google Messages.
Samsung is removing its built-in messaging app in favour of Google; this project gives users another choice.

## Features

- Native Android SMS/MMS default-handler implementation
- Material 3 UI with the Fixd brand palette (indigo / violet / gold)
- Built-in software keyboard (`FixdKeyboardService`) with letters, symbols, shift, backspace, return
- Bundled `androidx.emoji2` for full color emoji on every device, plus an Android system spell-checker integration
- Send and receive SMS, including multipart messages
- Conversation list, thread detail, compose screen, settings
- JSON backup of all messages to `Downloads/fixd-messaging-backup.json` with deduplicated restore
- Adaptive launcher icon and notification icon
- Light and dark theme, Android 12+ dynamic-color opt-in

## What still needs work

- MMS PDU send/receive pipeline (the receiver registers, but `MmsSender` is a TODO that needs full m-send-req PDU encoding)
- A custom in-app spell-correction strip backed by `SpellCheckerSession`
- Picture / attachment composer UI for MMS
- Group threads and contact resolution from the system Contacts provider

Contributions welcome - see `docs/architecture.md` and `docs/mms.md` for design notes.

## Building

The app is a standard Android Gradle project (Kotlin 2.0.21, AGP 8.6.1, JDK 17, compileSdk 34).
The Gradle wrapper is generated by CI but you can also generate it locally:

```bash
gradle wrapper --gradle-version 8.10.2 --distribution-type bin
./gradlew :app:assembleDebug
```

The debug APK is produced at `app/build/outputs/apk/debug/app-debug.apk`.

## Getting the APK from CI

Every push to `main` runs `Android CI`, which builds the debug APK and uploads it as a workflow artifact named `fixd-messaging-debug-apk`.
Download it from the [Actions tab](https://github.com/fixdfun/fixd-messaging/actions) of the latest successful run.
Unzip the artifact and sideload `app-debug.apk` onto a device with USB debugging enabled:

```bash
adb install app-debug.apk
```

Then open Settings -> Apps -> Default apps -> SMS app and pick **Fixd Messaging**.

## Privacy

This repository contains no personal or business data from Fixd.fun other than the brand colors and logo concept.
Messages, contacts and any other user data live only on the user's device.
